# 03. Datatypes

Any variable created in Python has its corresponding datatypes such as integer, float, boolean, string, tuple, dictionary, list, set etc. Datatype determines what kind of operation can be performed with them. For instance, two integers or two floats or integer with float and vice-versa can be divided. E.g., 4/2, 5.6/2.0, 5/2.0, or 9.0/3 are possible, but "World"/"MAn" is not considered a valid operation.
