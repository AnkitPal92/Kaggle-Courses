# 01 Arithmetic and Variables
## Printing statements.
Priniting statements is one of the simplest operation in computer programming. In Python, within the 'print' function we put inside double inverted commas the statement to be printed. The statement is then treated as a string datatype and the 'print' function returns the statement as it is.
## Arithmetic Operation.
Amongst arithmetic operations addition, subtraction, division, multiplication, modulus, floor division and exponentiation are some primary operations that can be computed within the print function directly by putting the arithmetic expression inside single inverted commas or by assigning the arithmetic expression to a variable and then printing that variable.
## Comments.
Comments are annotations which begin with a pound sign(#) as the first character which help others to understand the our codes. Moreover, they help us revisit and recollect the codes quickly, especially when we look at the codes after a hiatus. Further, while writing a very long program it comes very handy in summerizing the function of each code block.
## Variables.
Variables are essential to save the result of an operation. For instance, Earlier, in arithmetic operation, we noticed an alternative way to print the result by assigning the arithmetic expression to a variable and then print the variable. That way the value of the arithmetic operation is stored in the variable.

Variables are created by directly assigning values against them to the rightside of an assignment operator(=). E.g., test_var = 4 + 5, where test_var is the variable. Similarly, moe than one variable can be created whenever required.

While manipulating variables, the value of a variable can always be overrided by another value. The variable only stores the last value assigned to it, and does not save previous values.
